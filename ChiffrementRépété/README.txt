
Titre : Automatisation is key 

Description : Vous avez reçu un message étrange qui a été encodé plusieurs fois. Votre mission consiste à décoder le message et à trouver le message caché à l'intérieur. Pour vous aider, nous vous avons donné un fichier nommé "cipher.txt" qui contient le message encodé. Bonne chance !
La réponse sera nécessaire pour la suite. 


Indices :
Le message encodé a été encodé en base64 plusieurs fois.
Essayez de décoder le message plusieurs fois jusqu'à ce que vous trouviez le message caché à l'intérieur.
Bonne chance pour cette première épreuve de cryptographie répétée !
